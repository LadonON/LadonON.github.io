# LadonON GitHub Site

Static GitHub Pages site for `LadonON`.

## Files

- `index.html` - Site markup
- `styles.css` - Styling and responsive layout
- `script.js` - GitHub API fetch + project rendering
- `404.html` - Basic fallback redirect

## Deploy on GitHub Pages

1. Create or use the repository `LadonON.github.io`.
2. Copy these files to the root of that repository.
3. Push to the `main` branch.
4. In repo settings, ensure Pages is set to deploy from `main` branch root.

The site will then publish at `https://ladonon.github.io/`.
